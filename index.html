FLOSS-POS by maxolasersquad

POS System on a LAMP stack

FLOSS-POS is a point of sales system that runs on a LAMP stack. This project is a fork of the IS4C software, which can be found at http://www.wedge.coop/is4c/

Dependencies

apache2 mysql-server-5.0 mysql-client-5.0 php5 php5-mysql openssh-server openssh-client build-essential

Install

To install the server, go into the directory that you saved the files in and run ./get-server.sh To install a lane, go into the directory that you saved the files in and run ./get-lane.sh
